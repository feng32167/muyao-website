# MU YAO Personal Website

这是一个使用HTML、CSS和JavaScript构建的个人网站，具有以下特点：

- 彩虹霓虹灯效果的标题动画
- 响应式设计，适配移动端和桌面端
- 文章展示页面
- 社交媒体链接

## 技术栈

- HTML5
- CSS3 (动画、Flexbox、Grid)
- JavaScript
- 响应式设计

## 目录结构

```
.
├── index.html          # 主页
├── articles.html       # 文章列表页
├── css/
│   ├── style.css      # 主要样式
│   └── articles.css   # 文章页面样式
├── js/
│   └── script.js      # JavaScript脚本
└── images/            # 图片资源
```
